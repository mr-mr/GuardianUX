The setup

All html files are built up in Codekit .kit files. And you'll need 5 applications

  • Codekit
  • Mamp
  • Github app for Mac
  • A text editor
  • An FTP editor (leave for now)

As everything is setup on github, github itself has a html preview pane (although quite buggy)

Framework

The library framework consists of all components we need to create our prototypes. Each person can take their own part, but focussed into a single page plit into an explanation and code equivalent.
